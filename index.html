Download as .zip View project on GitHub
  • Simulation inspired by agar.io one of the most amazing javascript games on the web. Play the original here
  • See list of game interactions below

Interactions

• Player grows when food pebbles are picked up.

• Player slows down a bit as you grow larger.

• Ridged border on the player changes color to reflect the color of the last pebble eaten.

• Food pebbles on screen shrink a bit as the player grows to simulate zooming out.

• Food pebbles actively try to avoid the player.

• Grid background scales with player size to simulate zooming out to accommodate player size.

• Game over screen displays when player reaches a certain size. Prevents the sim from getting out of control.
